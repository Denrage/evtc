Namespace GW2Scratch.EVTCAnalytics.Model.Agents

Classes

Agent

Represents an agent in the game - an entity with health, capable of using skills and movement in some cases.

AgentOrigin

Contains metadata of the origins of an Agent from raw log data. Some Agents might be multiple merged raw agents and this class allows identifying these cases.

AttackTarget

Represents an attack target of a Gadget. These typically appear as targetable crosshairs in the game.

Gadget

Represents a gadget in the game. Gadgets are a special type of NPCs that are immobile and can be damaged through AttackTargets.

NPC

An NPC is a non-player Agent. An NPC may be hostile or friendly depending on its team, they are capable of movement, casting skills and even of entering downed state in rare cases.

OriginalAgentData

Represents the original identification values of a ParsedAgent used in construction of an Agent.

Player

Represents a character of a player.

Enums

EliteSpecialization

Represents an Elite Specialization of a Profession in the game. A player may only have one of these at a time and they are tied to specific Professions. Characters

Profession

Represents a profession of a player character in the game.
